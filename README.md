# Smart Resource Manager : Allocation and Utilization Suite

## Description
Smart Resource Manager is a web-based application designed to manage the allocation and utilization of resources efficiently.  
The main goal of this project is to help users plan, track, and optimize resources through a simple and interactive interface.

This project is created using HTML and JavaScript and is suitable for beginners learning frontend development and JavaScript logic.

## Features
- Resource allocation management
- Resource utilization tracking
- Simple and user-friendly interface
- Dynamic functionality using JavaScript
- Basic responsive design

## Technologies Used
- HTML
- JavaScript

## Project Structure
Smart-Resource-Manager/
- index.html
- data.js
- README.md

## How to Run the Project
1. Download or clone the repository
2. Open the index.html file in any web browser
3. Use the application to manage resources
